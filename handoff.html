CLI command
Create an H100 VM via the Crusoe CLI:
crusoe compute vms create \ --name my-vm \ --type h100-80gb-sxm-ib.8x \ --location us-southcentral1-a \ --image ubuntu22.04:latest \ --keyfile ~/.ssh/id_ed25519.pub

Pick an L40S or A100 GPU in us-southcentral1-a to validate your stack. Use crusoe compute vms types to list all available GPU types, and crusoe locations list to see regions. Stopped VMs are not charged for on-demand usage. If no image is specified, the system defaults to Ubuntu 22.04.

Generate a key pair
Run this command in your terminal to generate an Ed25519 key pair — then add the public key here.
ssh-keygen -t ed25519 -C "your@email.com" # Then copy the public key: cat ~/.ssh/id_ed25519.pub
Using your key
After adding your key here, select it when creating an instance. You'll be able to SSH in immediately once the VM is running:
ssh ubuntu@<instance-ip>
