Persistent block storage for your virtual machines. Attach and detach across instances — data persists independently of VM lifecycle. Supports snapshots and resizing up to 10 TiB.
Visit disk docs
Create & attach a disk in two steps
Create your disk, then select an instance to attach it to.
1
Create disk
Choose a name, size (1 GiB–10 TiB), and a location that matches your instance.
2
Attach to instance
Select an existing instance in the same location. The disk will be available as a block device immediately.
Or create via CLI:
crusoe storage disks create \
  --name my-disk \
  --size 500GiB \
  --location us-east1-a
💡 Create your disk in the same location as your instance to avoid cross-region latency.

Generate a key pair
Run this command in your terminal to generate an Ed25519 key pair — then add the public key here.
ssh-keygen -t ed25519 -C "your@email.com" # Then copy the public key: cat ~/.ssh/id_ed25519.pub
Using your key
After adding your key here, select it when creating an instance. You'll be able to SSH in immediately once the VM is running:
ssh ubuntu@<instance-ip>
